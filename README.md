# BSG Game System

## Getting started

### Turso DB
Turso is cheap, free, and fast.

Sync/Create migrations
```bash
pnpm run db:generate
```

Apply migrations to DB
```bash
pnpm run db:migrate
```

View DB in UI
```bash
pnpm run db:studio
```
